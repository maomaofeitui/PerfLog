This is a sample stand alone test class that shows how to use PerfLog and PerfLogAppLogger APIs within 
a J2SE application. The application includes perfLog.properties that is set to use 
Standard Out for performance logging.

To run this stand alone application, execute the runnable jar 

	PerfLogTestJavaApp.jar 

from the bin directory in the distribution as follows:

java -classpath PerfLogTestJavaApp.jar perf.log.test.TestApp
