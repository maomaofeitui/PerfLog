These samples use PerfLog.jar and PerfLogAppLogger.jar to demonstrate PerfLog framework.

PerfLogTestWebApp - is a web application that calls JAX RPC and JAX WS web service application
This application uses HelloWorldJaxRpcWebServiceProject and HelloWorldJaxWsWebServiceProject
  deployed in a different JVM
HelloWorld - is a Java Bean that is exposed as a JAX RPC and JAX WS Web Service


PerfLogTestJavaApp - is a stand alone Java application

For more details refer to the docs section
